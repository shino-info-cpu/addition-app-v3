# v3 業務イメージ整理表

これは実際の SQL テーブル設計ではなく、
「頭の中ではこういう Excel シートっぽく見えたい」という整理用の資料です。

- 実DBの正規化はいったん無視
- まずは業務上どう見えたら分かりやすいかを優先
- 列名は全部日本語

## この資料の使い方

- まずはこのファイルで「どう見えたら分かりやすいか」を決める
- そのあとで必要なら SQL に分解する
- つまり、これは DB 定義書ではなく、業務の見え方の整理表
